Class Tracks

Follow these classes in order. Each class has a goal, clear examples, and mini checks.

Class 1: Number Skills

Integers, positives/negatives, order of operations, and mental math habits.

  • Compare and order numbers
  • Use PEMDAS without panic
  • Check answers quickly

Class 2: Fractions, Decimals & Percents

Convert between forms and solve real-life percentage problems.

  • Fraction simplification
  • Percent increase/decrease
  • Word problem strategies

Class 3: Variables & Expressions

Understand what variables mean and how to evaluate expressions.

  • Translate words to expressions
  • Substitute values for variables
  • Combine like terms

Class 4: Equations Basics

Solve one-step and two-step equations using balancing logic.

  • Inverse operations
  • Equation checking
  • Simple equation word problems

Class 5: Graphing & Linear Equations

Read coordinate graphs, use slope-intercept form, and identify patterns.

  • Plot points correctly
  • Understand slope and intercept
  • Model real-life trends

Class 6: Algebra Confidence Lab

Mixed review, challenge sets, and progress feedback to lock in skills.

  • Target weak topics
  • Timed confidence drills
  • Final readiness checklist
